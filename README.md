# cft_test
Тестовое задание для ЦФТ  
Инструкция по запуску:  
В командной строке набрать   
java -jar <путь к cft_test.jar> <имя файла с входными данными> <имя файла с выходными данными> <тип данных> <тип сортировки>  
Типы данных:  
-i - целые числа  
-s - строки  
Тип сортировки:  
-a - по возрастанию  
-d - по убыванию
